# Task1
 Это интерактивное приложение для создания геометрических фигур, таких как круги, эллипсы и прямоугольники. Пользователи могут настраивать размеры, цвета и размещение фигур на холсте. Приложение поддерживает одновременное рисование нескольких фигур и отображает название последней нарисованной фигуры, делая процесс рисования простым и увлекательным.
